# FAIRagro_DMP_Template

This [RDMO](https://rdmorganiser.github.io/) template leverages the [DMP4NFDI Template Framework](https://doi.org/10.5281/zenodo.16737078) and was developed collaboratively by [FAIRagro](https://fairagro.net/) and [DMP4NFDI](https://dmp.services.base4nfdi.de/) to provide tailored guidance and resources for agrosystem research.

The template is available for use directly in RDMO via the [FAIRagro DMP Service](https://rdmo.fairagro.net/), hosted by DMP4NFDI. It includes customized option sets and help text for questions tailored towards agrosystem research. It is meant to support users in creating a DMP as efficiently and easily as possible. It will be updated based on user feedback in the future. 

Furthermore, it can be imported to other RDMO instances through the xml file provided in this repository.

For feedback on the template please contact the [FAIRagro Helpdesk](mailto:dataservice@fairagro.net) or [create an issue](https://github.com/fairagro/FAIRagro_DMP_Template/issues/new) in this Github repository.
